SIGA para Alunos – Gestão de Configuração

Este repositório reúne os artefatos produzidos para a atividade de Gestão de Configuração de Software, aplicada ao desenvolvimento simulado do sistema SIGA para Alunos. O objetivo é demonstrar práticas essenciais de organização, controle de versões e registro de mudanças utilizando Git e GitHub.

Conteúdo do Repositório

Documentação do projeto

Itens de configuração definidos na atividade

Procedimentos de controle de versão

Processo de controle de mudanças

Registro das alterações e evolução do projeto

Objetivo da Atividade

Aplicar, em ambiente acadêmico, os conceitos fundamentais de gestão de configuração, mantendo rastreabilidade, organização e padronização dos artefatos produzidos durante o desenvolvimento do projeto.
